Spring Boot 3 with ChatGPT demo
===============================

# Features

* Spring HTTP Interface to ChatGPT
* Reactive support

# Get started

* Modify `application.properties` to set your OpenAI API key or set the environment variable `OPENAI_API_KEY`
* Run `mvn spring-boot:run`
